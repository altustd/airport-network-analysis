# The Architecture of Flight

*A Network Analysis of the US Airport System*

Graph-theoretic analysis of the US domestic airport network: degree distribution, betweenness centrality, small-world properties, and robustness under random failure versus targeted attack. Built with NetworkX, Plotly, and Quarto.

## Run Locally

```bash
pixi install

pixi run render-pdf  # PDF → docs/
pixi run preview     # live preview
```
